# Code editor - React.js

Clon de la famosa herramienta CodePen para probar y mostrar fragmentos de código HTML, CSS y JavaScript creados por el usuario. Funciona como un editor de código en línea y un entorno de aprendizaje de código abierto, los desarrolladores pueden crear fragmentos de código y probarlo.

## Herramientas
* Create React App => [Create React App](https://github.com/facebook/create-react-app).  

## Para ejecutar esta aplicación:
* Clonar el repositorio: 'https://github.com/cmurestudillos/codepen.git'
* Hay que tener instalado [npm](https://www.npmjs.com) y [node](https://nodejs.org/es/)
* Siatuarse en el directorio raiz del proyecto e instalar las dependencias: `npm install`
* Ejecutar el siguiente comando para arrancar la aplicación: `npm start`
